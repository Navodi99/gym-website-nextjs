# Gym Website (Next.js)

## Project Name
**gym-website-next**

---

## Tech Stack
- Next.js
- Tailwind CSS
- Figma
- Framer Motion
- EmailJS

---

## Project Overview
A modern, responsive gym website built with Next.js. It features smooth animations, clean UI design, and a functional contact form integrated with EmailJS.

---

## Setup Instructions

### 1. Clone the Repository
```bash
git clone https://github.com/Navodi99/gym-website-nextjs.git
cd gym-website-next
npm run dev

```
